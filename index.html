Vietnamese Flashcards

It might take a moment before the first card loads, be patient! Please click Next to load a new card; Previous to go back to the last card once the first card has loaded.

This is a flashcard app for Vietnamese words and phrases, all derived from the content of our courses. I've tried my best to gather everything from the courses, and all are thrown in the mix here. A few caveats below:

  • All flashcards are fully random! You might see the same card more than once, and there's no intelligence behind what's picked.
  • There might be (and likely are) spelling mistakes, and issues with diatrics. Do flag those so they can be fixed, or even better, you can fix them in the sheet! See below for more.
  • There's technically no offline functionality, but once loaded, the page should work without internet.
  • If you want to add content to the flashcards, please request access to the following sheet, which functions as the database for the app.
